# Online Banking System (Spring Boot + JSP + MySQL)

## Overview
A complete online banking system built with Spring Boot, JSP, MySQL, and Bootstrap.

This project supports:
- User login & registration
- Account creation
- Dashboard (Admin + User)
- Transfer money between accounts
- Deposit / Withdrawal
- Transaction history
- Add beneficiary
- Profile management

## Tech Stack
| Layer | Technology |
|-------|------------|
| Backend | Java 17, Spring Boot |
| Frontend | JSP, HTML, CSS, JS, Bootstrap |
| Database | MySQL |
| Build Tool | Maven |

## How To Run
1. Clone the repo
2. Import as Maven project in IntelliJ / STS / Eclipse
3. Create database `online_banking` in MySQL
4. Import `online_banking.sql`
